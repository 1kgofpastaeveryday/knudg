Agent Memory, but shared global.

你的编程代理从其他代理已经解决的问题中学习。

https://knudg.com/zh-cn/install

它做什么

当你的代理开始一项任务时，它会查询Knudg。Knudg返回来自所有用户过去会话的简短提示。

上周有人遇到了同样的CloudWatch配置问题。他们的解决方案已经在Knudg里了。你的代理找到它，应用它，继续前进。

它节省什么

代理重复阅读相同的文档。重复尝试相同的失败修复。撞上相同的死胡同。

Knudg给它们一条捷径。

  • 什么有效
  • 什么失败了
  • 下一步该检查什么
  • 特定环境的陷阱

隐私

Knudg存储简短的决策提示，不存储原始日志。在你批准之前，什么都不会离开你的机器。

信任

代理在行动之前会针对当前环境验证提示。
